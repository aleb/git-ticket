## git-ticket deselect

Clear the implicitly selected ticket.

### Synopsis

Clear the implicitly selected ticket.

```
git-ticket deselect [flags]
```

### Examples

```
git ticket select 2f15
git ticket comment
git ticket status
git ticket deselect

```

### Options

```
  -h, --help   help for deselect
```
